# NightWatch Example

## folders

- bin: chrome driver
- env: 環境變數設定
- test: 測試程式碼
- nightwatch.conf.js: nightwatch設定檔
- package.json: npm設定檔

## test steps

1. 連至http://www.google.com.tw
2. 輸入關鍵字 "github"
3. 點擊第一個查詢網址
4. 輸入關鍵字 "bbandydd"
5. 點擊第一個連結

## setup

```
npm install
```

## run test

```
npm run start
```
