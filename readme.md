# NightWatch Example

## folders

- test: 測試程式碼
- nightwatch.conf.js: nightwatch設定檔
- package.json: npm設定檔

## setup

```
npm install
npm install -g nightwatch
```

## run test

Run main.js

```
npm run start
```

Run github.js

```
npm run github
```

Run all tests

```
npm run test
```

Run all tests on Firefox

```
npm run firefox
```

jshint

```
npm run jshint
```
